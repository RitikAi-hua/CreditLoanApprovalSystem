# 💳 CreditWise – Loan Approval Prediction System

## 📌 Overview

**CreditWise** is a Machine Learning-based classification system that predicts whether a loan application should be approved or rejected based on applicant financial and demographic details.

The goal is to assist financial institutions in making faster, consistent, and data-driven loan approval decisions.

---

## 🧠 Model Approach

* Supervised Classification
* Algorithms Tested: Logistic Regression, KNN, Naive Bayes, Decision Tree, XGBoost
* Best model selected based on Accuracy, Precision, Recall, and F1-Score

---

## ⚙️ Data Processing

* Missing value handling
* One-Hot Encoding for categorical variables
* Feature Scaling using StandardScaler
* Train-Test split with proper `fit()` and `transform()` to avoid data leakage

---

## 📊 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix
